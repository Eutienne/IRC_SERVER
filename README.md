# IRC_SERVER
This repo contains a cpp boilerplate ready to use with catch2 unitesting framework. It uses Cmake which is managed with a Makefile.

# How to run

Build the project and run the program :
```
make
cd build/apps
./helloWorld.out
```
Run unitests : 
```
make test
```
Clean everything :
```
make clean
```
